Requirements

  • WordPress 2.2 or 2.0.x/2.1.x with WordPress Widgets

Installation

  1. Download youtube-subscribe-widget.zip
  2. Extract and upload youtube-subscribe.zip to the wp-content/plugins/ directory.
  3. Enable YouTube Subscribe Widget in the Plugin admin panel.
  4. In widget admin panel: drag the widget to the sidebar and then edit options - enter the YouTube username and title.
