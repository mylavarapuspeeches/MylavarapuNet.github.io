Add a widget to display YouTube subscribe box in the sidebar.

== Description ==
This widget will display a YouTube SUBSCRIBE button, the number of subscribers and videos in the sidebar. When configuring you have to specify your YouTube username. You can also change the widget width and height as well as the widget title. The title could be linked to the YouTube profile as well.

This plugin is provided by [Flat Rock Technology](http://www.flatrocktech.com/). For more information please do not hesitate to contact us.

== Requirements ==

* WordPress 2.2+ or 2.0.x/2.1.x with WordPress Widgets

== Installation ==

1. Download youtube-subscribe-widget.zip
2. Extract and upload youtube-subscribe.zip to the wp-content/plugins/ directory
3. Enable YouTube Subscribe Widget in the Plugin admin panel
4. In widget admin panel: drag the widget to the sidebar and then edit options - enter the YouTube username and title.


== Frequently Asked Questions ==

* What is necessary to configure the widget?
Only the username of the yourtube profile.
